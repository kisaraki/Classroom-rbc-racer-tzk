# RBC Racer 階段結果報告

本目錄只存放已實際執行之開發階段結果報告。

命名方式：

```text
phase-00-report.md
phase-01-report.md
...
phase-10-report.md
```

每份報告必須依根目錄的 `PHASE_REPORT_TEMPLATE.md` 撰寫，並包含：

- 本階段授權與排除範圍。
- 實際修改內容與檔案。
- 自動測試及實際瀏覽器驗收結果。
- 發現的錯誤、重現步驟與原因。
- 修正方式及修正後重測。
- 效能證據與殘餘風險。
- PASS 或 BLOCKED 結論。

不得預先建立空白 PASS 報告，也不得在測試失敗或錯誤未修正時將階段標示為 PASS。
